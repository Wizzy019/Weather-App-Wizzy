# Weather App Wizzy 

**Weather App Wizzy** is a web application that allows users to search for the weather of any location by entering the name of a town or locality. It provides quick, easy-to-read weather information for the searched location.

## Key Features
- Search for weather by town or locality
- Displays current weather information
- Simple and responsive UI for easy access
- Completed project — fully functional
